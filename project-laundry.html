Laundry Management System

A full-stack system for managing laundry pickup, delivery, and order tracking. Built to make the process faster, easier, and more organized for both customers and staff.

GitHub Back to Projects
Year 2024
Type Capstone Project

OVERVIEW

Project Overview

Laundry Management System is a full-stack application that helps laundry businesses manage orders, schedule pickups and deliveries, track status, and monitor revenue. It provides a simple and efficient solution to replace manual record-keeping.

MY ROLE

  • Frontend Development
  • Backend Development
  • Database Design

  • UI/UX Implementation

KEY FEATURES

  • User Authentication (Customer & Admin)
  • Place and Track Laundry Orders
  • Schedule Pickup and Delivery
  • Real-time Order Status Updates
  • Admin Dashboard with Analytics

PROBLEM → SOLUTION

Problem: Laundry businesses relied on manual logs and text messages to manage orders, causing lost records and scheduling conflicts.

Solution: Built a system that automates order management, provides real-time tracking, and organizes business operations in one dashboard.

CHALLENGES

  • Implementing real-time updates using Firebase Firestore
  • Designing a clean and easy-to-use interface
  • Handling user roles and secure authentication
  • Structuring the database for efficient data management
