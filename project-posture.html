OVERVIEW

Project Overview

An AI-powered desktop application designed to help users maintain proper posture during prolonged computer use. The system uses a machine learning model trained through image classification to detect posture in real time via webcam.

When poor posture is detected for a set duration, the system triggers a sound alert to notify the user. This promotes awareness and helps prevent long-term health issues.
